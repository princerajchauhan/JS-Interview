JS Interview Question

Q1. Difference between “ == “ and “ === “ operators.

  • The == operator does the type conversion of the operands before comparison.
  • the === operator compares the values as well as the data types of the operands.
Example :-
            
            let a = 5
            let b = '5'

            console.log(a == b)    // true
            console.log(a === b)   // false

Q2. What are the differences between var, let and const?

  • var and let create variables that can be reassigned another value. const creates "constant" variables that cannot be reassigned another value
Example :-
            
            {
                var a =10;
                let b =20;
                const c =30;
            }
            console.log(a);  //10
            console.log(b);  // b is not defined
            console.log(c);  // c is not defined
            
            let d = 10;
            let d = 20; // SyntaxError: Identifier 'd' has already been declared
            d = 11;
            console.log(d); // 11
                
            const e = 10;
            e = 11;
            console.log(e); // Assignment to constant variable

Q3. What is hoisting?

  • In JavaScript, hoisting allows you to use functions and variables before they're declared.
  • Hoisting only possible for var not for let and const.
Example :-
            
            console.log(h);     // undefined
            var h = 'hello';

Q4. What is a Temporal Dead Zone?

  • The reason that we get a reference error when we try to access a let or const variable before its declaration is because of the temporal dead zone (TDZ).
Example :-
            
            console.log(h); // ReferenceError because we're in the TDZ
            let h = 'Hello';  // ReferenceError: Cannot access 'foo' before initialization
